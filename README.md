# cpu scheduling  Project  

## How to build and run
Compile and run by typing the following commands:

```bash
	$ cd 
	$ make 
	$ make run
```

### Genreate test files
Test files can be generated from a normal distribution.
```bash
    $ bin/data_gen.py -h
    $ bin/data_gen.py .... 2>/dev/null > data/new_data.dat
```
